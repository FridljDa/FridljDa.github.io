# Conclusion
- Some high level concepts are still relevant
- Text advertising Google products like agent2agent protocol and Gemini ADK can safely be skipped. Here, a bias by the author, who is a Google employee, is evident.
- Code examples based not very useful. The text descriptions suffice to understand the concepts. I personally like pydantic ai agent framework more than the ones used in the book (langchain, google adk, agent crew). But overall,, the code examples are not very useful, and in some cases, even distracting.
